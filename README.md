# 동그리미 프로젝트

직장 기반 주거 지역 추천 서비스 프로젝트입니다.

## Teck Stack

| Category            | Tool           |
| ------------------- | -------------- |
| **Framework**       | Next.js 14.1.0 |
| **Language**        | TypeScript     |
| **Package Manager** | Yarn           |
| **Bundler**         | Webpack        |
| **Version Control** | Git, GitHub    |

## Getting Started

```bash
yarn install
```
